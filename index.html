Kitchen API - Product roadmap
Go to project

This is a product road map created during the course “Technical Product Manager” from Platzi.

It is for an API "Kitchen API" that will be used by the RoloFoods App (Similar to Rappi), It is divided in 3 phases and each of them in 2 or 3 sprints.

The idea is to create a set of endpoints to update ingredients and dishes offered in the customer App according to their availability, so that the App removes unavailable dishes dynamically and from available dishes it can remove extra ingredientes, like avocado, sauces, etc.
